Description de la base de données Mongo DB :
- Le nom de la base de données est "simpleblog". Elle contient une collection nommée "article" stockant tout les posts du site. Un article possède un "titre", un "auteur", une "date" et un "resume".


Le projet est clé en main, il faut lancer la commande "node .\server.js" et ce connecter avec un navigateur sur l'adresse localhost:8000.
